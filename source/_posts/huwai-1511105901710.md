# 部署
可以把整套服务起在docker中，下面没有涉及mysql，可以通过--link mysql_server单独到一个mysql docker容器；
在docker中取下三份代码；

- koa2 唯一需要起到服务代码
提供服务接口和手机页面静态资源服务
		
        npm install && npm run compile	
        pm2 start pm2.json

- hw 中
提供手机页面静态资源
		
        npm install && npm run build:prod	
        cp dist ../koa2-hw/public
       
- hw-admin
提供pc管理后台静态资源
		npm install && npm run build
        
- nginx
提供手机页面和管理后台入口
